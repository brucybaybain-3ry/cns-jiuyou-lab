# cns-jiuyou-lab

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容互不关联，每个页面都可以作为独立的静态资源访问。

仓库本身不针对任何特定域名或网站，仅作为 HTML 文件的集合与版本管理空间。

## 目录说明

```
/
├── pages/          # 存放各独立 HTML 页面
├── assets/         # 页面所需的公共资源（样式、脚本、图片等）
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹或文件代表一个独立的页面，命名以页面主题或功能为参考。
- `assets/` 目录存放多个页面可能共用的资源文件，便于统一维护。
- 若页面有专属资源，建议放置在该页面所在目录内。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可直接在浏览器中打开运行。
- 页面之间无依赖关系，可单独部署或分发。
- 归档内容不保证时效性，仅反映上传时的状态。
- 所有页面均不包含跟踪、分析或第三方请求（除非在页面源码中明确标注）。

## 维护说明

- 本仓库由维护者不定期更新，新增页面或修正已有内容。
- 不承诺固定的更新频率或技术支持。
- 若发现页面显示异常或链接失效，可通过 Issues 反馈，但不保证及时修复。
- 不接受主动提交的 Pull Request 用于新增页面，但欢迎对已有页面进行 bug 修复的建议。

## 许可

本仓库内容按原样提供，不附加任何明示或暗示的保证。如需引用或分发，请自行评估适用性。
